Length
The Length() method returns the length of the string
Replace
It Replace a part of given string with another string.

It Concat or Joins the two or more strings together

It extracts a part of string based on given starting and ending index and returns a new string

For Example :- str = "helloworld" Result :- The length of string is 10

Slice Method

Get the String from the given start or end Index
For Example :- String = "helloworld" ,StartIndex = 3 , EndIndex = 7 Result :- lowo

Upper Case Method

Convert String to UpperCase

For Example :- string = "helloworld" character = l Result :- 2

Substring Method

Get the part of string from the given string
For Example :- String = "helloworld" , number = 45 Result :- 55

CodePointAt Method

Find the unicode of the character from the given String
