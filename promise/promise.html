Promise All Method

Defination :- It takes an iterable of promise and returns a single promise when all the promises fullfill.
Example :-
const promise1 = Promise.resolve(3); const promise2 = 42; const promise3 = new Promise((resolve, reject) => { setTimeout(resolve, 100, 'foo'); }); Promise.all([promise1, promise2, promise3]).then((values) => { console.log(values); });
Result :-
Array [3, 42, "foo"]

Promise Any Method

Defination :- It takes an iterable of promise and returns a single promise when any of the promises fullfill. promises fullfill.
Example :-
const promise1 = Promise.reject(0); const promise2 = new Promise((resolve) => setTimeout(resolve, 100, 'quick')); const promise3 = new Promise((resolve) => setTimeout(resolve, 500, 'slow')); const promises = [promise1, promise2, promise3]; Promise.any(promises).then((value) => console.log(value));
Result :-
"quick"

Promise Race Method

Defination :- It takes an iterable of promise and returns a single promise which compete first among all the proises
Example :-
const promise1 = new Promise((resolve, reject) => { setTimeout(resolve, 500, 'one'); }); const promise2 = new Promise((resolve, reject) => { setTimeout(resolve, 100, 'two'); }); Promise.race([promise1, promise2]).then((value) => { console.log(value); // Both resolve, but promise2 is faster });
Result :-
"two"

Promise AllSettle Method

Defination :- It takes an iterable of promise and returns an array of object with status and value when all the promise settle
Example :-
const promiseOne = new Promise(function (resolve, reject) { const x = 1; const y = 1; if (x === y) { resolve(4); } else { reject(); } }); const promiseTwo = Promise.reject(5) const promiseThree = new Promise((resolve,)=>{ setTimeout( ()=>{ resolve("hello") },100 ) }) let promises = [promiseOne,promiseTwo,promiseThree] Promise.allSettled(promises).then((results) => console.log("all Settled" ,results) );
Result :-
all Settled [ { status: 'fulfilled', value: 4 }, { status: 'rejected', reason: 5 }, { status: 'fulfilled', value: 'hello' } ]
